CoDAL

Command Line Switches

Switch Description Required
/S Name of the SQL Server Yes
/D Name of the database Yes
/U SQL Server username. If this is not provided, then Windows authentication is assumed. No
/P SQL Server password. If this is not provided, then Windows authentication is assumed. No
/N The namespace to use for generated classes. If this is not provided then the database name is used. No
/GP The path to generate C# classes. Yes
/SP The path to generate C# skeleton classes. The skeleton classes are generated once, after which they will not be overwritten. If no value is specified, then the skeleton classes are saved in the same folder as the other generated classes. No
